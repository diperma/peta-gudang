# Peta Sebaran Pembangunan Gudang

Visualisasi interaktif sebaran lokasi pembangunan gudang di seluruh Indonesia.

## Fitur
- Peta interaktif dengan marker clustering
- Data dimuat dari file Excel (xlsx)
- Popup detail untuk setiap lokasi gudang
- Statistik total lokasi dan cakupan provinsi

## Teknologi
- Leaflet.js untuk peta interaktif
- MarkerCluster untuk clustering
- SheetJS untuk membaca file Excel
- TailwindCSS untuk styling
